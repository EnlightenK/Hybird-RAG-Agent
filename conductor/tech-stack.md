# Technology Stack

## Core
- **Programming Language:** Python 3.10+
- **Agent Framework:** Pydantic AI
- **Database:** PostgreSQL with pgvector (for Vector and Semantic Search)

## Ingestion & Processing
- **Document Processing:** Docling (Support for PDF, Word, PowerPoint, Excel, Audio)
- **Embedding Generation:** Google Gemini (Generative AI Embeddings)

## Interface & Tooling
- **CLI Framework:** Rich (for terminal formatting and streaming)
- **Package Manager:** uv (fast dependency and virtual environment management)
- **Async Driver:** asyncpg (High-performance PostgreSQL driver)
